HTML , CSS and CSS Bootstrap.

Html

Html stands for hypertext markup language. Hypertext representing that one page links to another , markup representing that each code or action a developer writes is marked up using various element tags and language respresenting that it follows a certain sytax to work. It is primarily use to create the basic layout of a web page as in the skeleton of the webpage.

Css

Css stands for cascading style sheet. It is a language which can either me inline, internal or extenaml in the file. After html it is used to provide the eye-comforting layouts and designs to the webpage. Along with that it is also used to make the web page render properly on different screens.

Css Bootstrap

CSS Bootstrap is the most popular framework on github. It is a collection of codes written by other programmers and developrs for other developers to use while making their applications, making it easy for the producing developer to write their desired code quickly and more efficiently. Along with css the bootstrap also provides snippets for javascript.
